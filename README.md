# Intergalactic Call Center

## Comienzo

Clonas el repo en tu equipo y estas listo para comenzar.

### Prerequisitos

Tener Java 1.7+ y Maven instalado en tu equipo


### Instalación

Descargas el proyecto y usas:

```
mvn install
```
